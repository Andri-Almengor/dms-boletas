# Visibilidad de boletas finalizadas

- Los administradores pueden consultar todas las boletas finalizadas.
- Los administradores pueden filtrar el historial por técnico asignado.
- Los técnicos solamente pueden consultar boletas finalizadas en las que figuren como asignados o que hayan creado.
- La regla se aplica a la lista, el detalle, la firma y las evidencias.
- Los estados históricos `FINALIZADO` y `FINALIZADA` se interpretan como el mismo estado.
