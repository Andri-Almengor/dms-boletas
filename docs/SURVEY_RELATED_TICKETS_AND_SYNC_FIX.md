# Encuestas de boletas relacionadas y sincronización

- El detalle administrativo de una encuesta identifica cuando corresponde a varias visitas.
- Muestra la cantidad de boletas, todos los consecutivos y la información de cada visita.
- Cada boleta conserva un acceso directo a su detalle.
- La finalización ya no se bloquea por el prefijo de un identificador creado offline.
- El bloqueo depende únicamente de operaciones reales pendientes en la cola de sincronización.
