# Visitas relacionadas y plantilla de boletas

Las visitas relacionadas conservan la plantilla oficial de boleta configurada en `TEMPLATE_BOLETA_ID`.

- Cada visita genera su propio documento y PDF con la misma plantilla usada por una boleta individual.
- El correo del seguimiento agrupa y adjunta esos PDF individuales.
- La encuesta y el enlace de firma siguen siendo únicos para todo el seguimiento.
- No se sustituye ni se rediseña la plantilla original.
